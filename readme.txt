--- README ---   

Per avviare il programma è necessario eseguire i seguenti comandi su shell:

cc Avvia.c -o avvia
./avvia [MOD_AVVIO] [PATH_DATASET]

[MOD_AVVIO] deve essere sostituito con: "NORMALE" oppure "FALLIMENTO"

[PATH_DATASET] deve essere sostituito con il percorso del file dataset.csv

Una modalità diversa da quella specifica o un percorso non esistente genererà un errore.
